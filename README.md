# Relaxometry and Diffusion MRI Reveal Sex Differences in the Association Between Daytime Sleepiness and White Matter Myelination

This repository contains the necessary data and analysis code for the Sleep Advance paper titled *"Relaxometry and Diffusion MRI Reveal Sex Differences in the Association Between Daytime Sleepiness and White Matter Myelination"*.

## Data

Data are uploaded here and in the journal supplementary material.
